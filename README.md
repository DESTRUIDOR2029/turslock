# 🌾 TursLock

Plataforma web gratuita que conecta produtores rurais a turistas,
incentivando o turismo rural sustentável e o desenvolvimento econômico
de comunidades do campo.

Projeto desenvolvido no 1º semestre de Ciências da Computação,
alinhado aos Objetivos de Desenvolvimento Sustentável da ONU
(ODS 8 — Trabalho Decente e Crescimento Econômico e ODS 11 — Cidades
e Comunidades Sustentáveis).

---

## 🚀 Funcionalidades

- Vitrine de produtores rurais e experiências disponíveis
- Carrossel de imagens responsivo (sem JavaScript)
- Layout adaptado para mobile e desktop
- Design moderno e acessível

---

## 🛠️ Tecnologias utilizadas

- HTML5
- Tailwind CSS
- CSS puro (animações e carrossel sem JavaScript)

---

## 🖥️ Como visualizar

1. Clone o repositório:
```bash
   git clone https://github.com/DESTRUIDOR2029/turslock
```
2. Abra o arquivo `index.html` no navegador.

---

## 🌍 Contexto

O TursLock nasceu da ideia de valorizar comunidades rurais por meio
do turismo, oferecendo uma plataforma simples e gratuita para que
produtores possam divulgar seus espaços e experiências.
